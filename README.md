# MiniRust-Discussion

Chào mừng các bạn sinh viên đến với diễn đàn thảo luận của môn học Thiết kế Trình biên dịch (MiniRust)!

## Hướng dẫn sử dụng:
1. Hãy sử dụng tab **Discussions** ở phía trên để đặt câu hỏi, thảo luận và hỗ trợ lẫn nhau trong quá trình thực hiện bài tập lớn.
2. Xem các thông báo chung từ Thầy/Trợ giảng trong danh mục **Announcements** của phần Discussions.
3. Tuân thủ nội quy môn học, không đăng trực tiếp lời giải hay code hoàn chỉnh của các bài tập lớn lên diễn đàn.

Chúc các bạn làm bài tốt!
